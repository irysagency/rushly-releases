# Rushly — Mises à jour

Canal de mise à jour **public** du plugin [Rushly](https://rushly-ai.app) pour Adobe Premiere Pro.

- **`manifest.json`** — version courante du plugin (lu au démarrage par la mise à jour in-app).
- **Releases** — le `.zxp` signé de chaque version (téléchargé automatiquement par le plugin).

### Installer Rushly
Télécharge l'installeur (`.dmg`) sur **https://rushly-ai.app**.
Ce dépôt ne sert qu'aux **mises à jour automatiques** du plugin déjà installé : quand une
nouvelle version est publiée ici, le plugin affiche « Nouvelle version disponible » et se met
à jour en un clic, sans réinstallation.
